# Web Mailing Service

Сервис управления рассылками на Django

## Технологии

- Python 3.13
- Django 6.0.5
- PostgreSQL (планируется)
- pytest, flake8, black, mypy

## Установка

git clone <url>
cd web_mailing_service
python -m venv venv
source venv/bin/activate  # или venv\Scripts\activate на Windows
pip install -e .[dev]
